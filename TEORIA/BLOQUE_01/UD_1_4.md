## üß© Sentencia if

La sentencia `if` permite **ejecutar un bloque de c√≥digo solo si una condici√≥n es verdadera**.

```python

if edad >= 18:
    print("Eres mayor de edad")
```

‚û°Ô∏è Resultado: `Eres mayor de edad`

üìå Si la condici√≥n no se cumple, el bloque no se ejecuta y el programa sigue su curso.

## üîÄ if...else

El `if...else` permite elegir entre dos caminos:

* Si la condici√≥n es verdadera, se ejecuta el bloque de `if`.
* Si no lo es, se ejecuta el bloque de `else`.

```python

if temperatura > 20:
    print("Hace calor")
else:
    print("Hace fr√≠o")
```

‚û°Ô∏è Resultado: `Hace fr√≠o`

## üîó if...elif...else

Cuando tenemos **m√∫ltiples condiciones**, podemos usar `elif` (else if).

```python
nota = 72

if nota >= 90:
    print("Excelente")
elif nota >= 70:
    print("Notable")
elif nota >= 50:
    print("Aprobado")
else:
    print("Reprobado")
```

‚û°Ô∏è Resultado: `Notable`

üìå El programa eval√∫a las condiciones en orden y ejecuta la primera que sea verdadera.

## üîÑ match-case (switch en Python)

En otros lenguajes se usa `switch`.
En Python, desde la versi√≥n **3.10**, tenemos `match-case`, que funciona de manera similar.

Ejemplo: clasificar un mes en su estaci√≥n correspondiente.

```python
mes = 4

match mes:
    case 12 | 1 | 2:
        print("Invierno")
    case 3 | 4 | 5:
        print("Primavera")
    case 6 | 7 | 8:
        print("Verano")
    case 9 | 10 | 11:
        print("Oto√±o")
    case _:
        print("Cualquier otro valor")
```

‚û°Ô∏è Resultado: `Primavera`

üìå Observa que en un mismo `case` podemos agrupar varios valores con `|`.

## üîé Comparativa: if vs switch (match-case)

| Aspecto                            | `if...elif...else`                                        | `match-case`                                      |
| ---------------------------------- | --------------------------------------------------------- | ------------------------------------------------- |
| **Disponibilidad**                 | Disponible en todas las versiones de Python.              | Disponible desde Python 3.10.                     |
| **Uso**                            | Ideal para condiciones complejas (`>`, `<`, `and`, `or`). | Ideal para comparar con valores exactos.          |
| **Legibilidad**                    | Puede volverse largo si hay muchas condiciones.           | M√°s limpio y ordenado para m√∫ltiples opciones.    |
| **Flexibilidad**                   | Permite condiciones l√≥gicas y expresiones.                | Est√° m√°s limitado a casos de igualdad y patrones. |
| **Equivalente en otros lenguajes** | if-else tradicional.                                      | switch-case (C, Java, JavaScript).                |

üìå En resumen:

* Usa **`if...elif...else`** cuando necesites condiciones complejas.
* Usa **`match-case`** cuando solo compares valores exactos y quieras un c√≥digo m√°s limpio.
