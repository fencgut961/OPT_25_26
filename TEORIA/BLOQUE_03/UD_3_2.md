## üìö Explicaci√≥n te√≥rica

### 1. ¬øQu√© es un m√≥dulo?

Un **m√≥dulo** es un archivo de Python que contiene funciones, clases y variables que se pueden reutilizar.

Python distingue entre:

* **M√≥dulos est√°ndar** ‚Üí vienen incluidos con Python (`os`, `sys`, `math`, etc.).
* **M√≥dulos externos** ‚Üí deben instalarse con **pip** (ejemplo: `requests`).

### 2. Formas de importar m√≥dulos


### 3. M√≥dulo `math`

Proporciona funciones matem√°ticas avanzadas.

Ejemplos:

* `math.sqrt(x)` ‚Üí ra√≠z cuadrada.
* `math.factorial(x)` ‚Üí factorial.
* `math.pi`, `math.e` ‚Üí constantes matem√°ticas.

### 4. M√≥dulo `os`

Permite interactuar con el sistema operativo.

Ejemplos:

* `os.getcwd()` ‚Üí devuelve el directorio actual.
* `os.listdir(ruta)` ‚Üí lista los archivos en una carpeta.
* `os.mkdir("nueva")` ‚Üí crea una carpeta.

```python

```

### 5. M√≥dulo `sys`

Permite obtener informaci√≥n del int√©rprete de Python.

* `sys.version` ‚Üí versi√≥n de Python.
* `sys.platform` ‚Üí sistema operativo.
